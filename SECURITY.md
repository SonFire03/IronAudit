# Security Policy

## Supported Versions

At this stage, only the latest `main` branch is actively supported.

## Reporting a Vulnerability

Please do not open public issues for sensitive security reports.

Use GitHub Security Advisories (preferred):
- Repository -> Security -> Report a vulnerability

Include:
- affected version/commit
- reproduction details
- impact assessment
- suggested mitigation (if available)

We will acknowledge reports and coordinate responsible disclosure.
